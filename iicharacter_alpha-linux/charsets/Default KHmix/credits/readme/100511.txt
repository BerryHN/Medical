・横結びウェーブ
使用フォルダ：hair_back，hair_back_accessory
　耳の上辺りで結んだツーテールをアクセサリパーツで作成しようかと作業を始めたんですけれど，どうにもバランスが取れなくて，結び目を上に上げて後ろ髪パーツとして作成しました。ウェーブをかけるのは当初の予定通りです。
作業用ファイルは「サイドツーテール」という名称にしていますが，後ろ髪では既に和さんの横結び，横結び(長)，yomoさんのゆるよこ結びがありますので，整合性が取れるようネーミングは横結びウェーブとしました。

　ウェーブが交差しているので髪の毛は１房ごとにレイヤーを変えています。それぞれ輪郭・基本色・ハイライトの３つのレイヤーを使っているので相当重くなってしまい，過去にレイヤを増やしすぎて何度もデータを飛ばした事もあってかなりおっかなびっくりの作業だったのですが，pixiaも新エンジン版の4.70になって安定しているようで，なんとか問題なく作業を終える事が出来ました。
