◆パーツについての注意

自分で使う時のために、体形差が欲しかったので、
デフォルトの素体１、素体２の他に、
UTAUユーザー互助会BBSの「キャラクター何とかゴースト作成依頼＆展示会場」にて
配布されていた男性用-試作素体(細身)、
及び、拙作の女性試作素体にあわせて作ったものがあります。
サイズが違うのでご注意下さい。

男性用-試作素体(細身)：作者不明。なんとか機ツバメ用燕尾服等内。
　　　　　　　　　　　　男性Yシャツ_aなどの1.09＋2.03組み合わせ素体に似せたものとのこと。
女性試作素体：拙作。おまけに同梱。
　　　　　　　素体１に素体３風の腕の長さと大きめの胸をつけたもの。
　　　　　　　浴衣の帯や柄などを素体１と一緒に使えるように作成。
　　　　　　　素体２風のポーズも作ったが、浴衣の場合、袖で全て隠れるので未使用。

◆このセットの改変・改良について
改変・改良は基本的に自由にしてくださってかまいません。
アクセサリの（服）とついているものはコピペして、
(瞳)のところを（髪）や（服）にするとカラー選択で色換えできるようになります。
その際は、accessory_backフォルダにも同名のパーツがあるので、コピペ・リネームしてください。
改変・改良パーツの配布の際は、元パーツとして記述されていれば、連絡は必要ありません。
透明に関しては名称の記述もいりません。


◆再配布について
以下の場合は再配布可とします。
・アップローダから流れ落ちてネット上から消えた場合
連絡は特に必要ありません。
