【今回のパーツ解説＆メイキングみたいなもの】
服装パーツ
・セーラー冬服1,2
　デフォルトのセーラー服の白い上着を襟と同色にしたものです。今まで既に誰かがやっていそうでやっていなかったものです。

アクセサリパーツ
・セーラー服用リボン(服)1,2,2s
　セーラー冬服を作成ついでにセーラー服用リボンの色変えパーツも作成してみました。ファイル名に(服)と付けていますが，基本色を黄色にしているため服とは違った色変化をします。
　2sはセーラー夏服2専用です。
　リボンをテストしていたところ，半袖だと長袖で隠れる部分が塗りつぶされていなくてその部分だけ赤くなってしまうので，急遽作成しました。
　
・もみあげ結びB(髪)
　前回予告した通り，太短いもみあげにばってん結びのもみあげを作成しました。短さにこだわったためにばってん結びの位置が上になってしまい，ばってんが見える前髪パーツがほとんどありませんのであしからず。

・肩載せ三つ編みC,D(髪)
　長い三つ編みの肩載せバージョン。１本三つ編みがCで，２本三つ編みがDです。
　今年の夏場にそうめんつゆのＣＭで女優さんがしていた１本三つ編みを見て，「この位置ならポーズ1と2で作り分けしないでもいける！」とか思っていたのですが，やっと作成出来ました。

　ついでに２本三つ編みのDも作成したわけですが，向かって左側の方はバランス的に背中に回した方がいいかな，そちら側が前に表示されない仕様にしてありますので，「表示されない」というクレームは無しの方向でお願いします。
（↑この結論出すまでかなり試行錯誤に時間かけてるんですよ…）
