後ろ髪パーツ
・ツインテール（ナチュラル）
・ツインテール（ナチュラル2）
アクセサリーパーツ
・ツインテール用カラーリボンY1
・ツインテール用カラーリボンY2

ふわっと広がった感じのツインテールです。
デフォルトの白リボンが付いているものが２，付いていないものが無印です。
ツインテールのシリーズはリボンが付いているものがデフォルトのようなので，ネーミングに悩みましたが，「ナチュラル」と付けた以上，リボンなしを無印としました。

アクセサリーパーツ
・肩載せ三つ編みA1
・肩載せ三つ編みA2
・肩載せ三つ編みB1
・肩載せ三つ編みB2

リクエスト板にもあった肩に載せた三つ編みです。
リクエストの方は耳の前で結んでそのまま垂らすイメージなのかもしれませんが…

身体の前に垂らすためにアクセサリーパーツとしてあります。
出来るだけいろんな衣装と組合せ可能なように，手の下を通るように配置しています。
使用可能な後ろ髪が限定されますので注意してください。

Aが２本結び，Bが１本結びで，1がポーズ1用，2がポーズ2用となっています。

Bはデフォルトの三つ編みのコピペ改変ですが，AはBから改変しようとしたところ，結局描き直す羽目になりましたｗ

パーツの位置は「accessory_middle_back」で良いかとも思ったのですが，胸元のアクセサリーパーツの上に表示されるように肩から下の部分は「accessory_front」，肩から上を「accessory_middle_back」としました。
