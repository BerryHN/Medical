アクセサリ
・ポニーテール各種
　ポニテだけで他の髪型と合わせたいという願望に基づき作製しました。
　斜めのが一番位置的に好きなので、反対側のもつくって擬似的にツインテもできるようにしてみました。

・もみあげ幅広３
　幅広２をひっくり返しただけですすいません。勝手にいじらせてもらいました。
　個人的にこっちのほうがしっくりきただけです。２も好きなんですけどね。

髪型
・ポニテ
　アクセサリ枠が５つしかないのでこっちでも普通のポニテを三つほど。アクセサリでいうと、斜めのものと同じ位置になります。
　ちなみにギザギザショートと安易に組み合わせていますので、そこが気になる人はアクセサリでベリーショートと組み合わせるとちょうどいいと思います。

頭
・耳なし
　犬耳とかつけるときに人間の耳が邪魔に思うことがあったのでつくりました。正直削っただけなので誰でもつくれるんですけど、特にあがって（と思う）のでつくってみました。ただ……２の方や青ざめたバージョンまでつくれるほどの技量はなかったということで。

・耳尖り
　エルフ耳ほど尖っていないけれど、少しは尖っていて欲しい耳が欲しかったのでやっぱりつくってみました。これも誰にでもつくれるものだと思います。あと上と同じく、普通の顔１をいじっただけです。

おまけについて
・白衣
　が入っています。はっきり言って駄作です。途中までつくってから右手のことに気づいたので無理してポケットを作製しました。ただ私が欲しかったからつくったというだけなので使い道はないと思ってください。あと見ての通り、白衣の下はキュロットスカートです……。だれかきちんとした白衣つくってくれないかなぁ。

お詫びと謝辞

全体的に雑です。しかも全てのパーツが何かの転用というか改変です。
他の作者様たちに申し訳ないのですが、使わせてもらいました。
あとこれは言い訳にすぎないのですが、マウス書きな上に影のつけかたとかわかんねとかいってる初心者なのでできれば大目に見てくださると嬉しいです。はい。
