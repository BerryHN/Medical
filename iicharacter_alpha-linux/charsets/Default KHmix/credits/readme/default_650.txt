キャラクターなんとか機用全身化素体パーツ集

デフォルトの女の子セットの全身化パーツ集です。
横300×縦650ピクセルの画像になります。
上方400ピクセルはデフォルトとパーツ位置が同じですので流用が簡単です。


◆使い方

解凍して出来たフォルダを、キャラクターなんとか機の
「data」フォルダの中にそのまま入れます。
その後、キャラクター選択で「default_650」に変更して下さい。

◆おまけ設定ファイル
エプロンドレス・サマードレス・セーター・巫女装束の組み合わせ例です
要デフォルトパーツです


タイツの上辺ラインはyomoさんのタイツ素材を参考にさせていただきました。
ローライズなボトムスに合わせるのに向いていると思います。




どうでもいいＱ＆Ａコーナー


Ｑ．どうしてデフォルトパーツも入っていなくて
素体セットとして完成されてない状態なの？

Ａ．せめておまけ設定ファイルがそのまま使えるようにしてたら、
ナローバンドすぎてうｐが途切れて出来なかったんだよ。
めげずに今後もナローバンド利用で生きていきます。


Ｑ．だから画像サイズもそのままなの？

Ａ．どっちかというと、これまでに作られたたくさんのパーツを
そのまま使いたかったから、足だけパーツメインなん…だ…

足元まできっちりデザインされてる服とかは
全身素体で新たに作るといいんじゃないかって思うよ。
それをこれまでのデフォルト素体セットに入れれば
300×400の腿から上画像にも使えるし。
