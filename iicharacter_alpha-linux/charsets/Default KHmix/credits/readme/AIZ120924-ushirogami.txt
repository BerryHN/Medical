　ダウンロードありがとうございます。今回は当初の想定よりも間が開いてパーツも溜め込んでいるため，いつになく長ったらしいので画面を最大化して読む事を推奨します。
　
【今回のパーツ解説＆メイキングみたいなもの】

後ろ髪パーツ
・縦ロール2a，2b，2c
使用フォルダ：hair_back

左右４本づつの縦ロール。
試作画像投稿時にも書きましたが，「ロリータ風味の縦ロール」を作りたいな，と思って作成したものの，なんか妙に大人びて見えたので試行錯誤していった課程です。
相当トライアルをしていますが，くるくるパターンを微修正とは行かずにパターンを弄るたびにほぼ全部を描き直しています。
途中放棄したものも含めて作業ファイルが10個，一度完成させたものの，テストしてみてから没にしたファイルもゴミ箱フォルダを開けてみたら２個ありました。
作業が面倒なので，向かって右側の輪郭線が仕上がった後にコピーして左右反転させて位置調整をして反対側の輪郭線としています。…おかげで上下位置がまずかったり，左右に拡がり過ぎたりとかで作業ファイルを途中で没にしているわけですが…

流れとしては
2a：「う〜ん，なんか意外に落ち着いちゃって全然ロリロリしていないような気がする。でもこれはこれでありだな。」
2b：「よ〜し，左右に膨らませて全体的なバランスで縦の長さの比率が小さくなるようにしてみよう。」→「う〜ん。それでもまだ大人びてるなぁ。没にしようかなぁ…でもこれはこれでありかな？」
2c：「やっぱ全体的に髪が下に下がっているのが大人びて見える理由かな？全体的に上に上げてみよう。」→「ダメだ。似たようなものにしかならなくてどうも上手くいかない。しかも2aと2bよりもバランス悪くなってる…orz（ここでいくつか作業ファイルを没）」→（この課程のどこかで下のロールから描いていたのを上のロールから描き始める）→「え〜い，もういっその事ロール間の間隔拡げてぱーっと上下にも拡げてみようか。」→「う〜ん，巻きが小さくなっちゃって下の毛先の位置が揃わないなぁ…こりゃ没かな？でもここまで出来たから一応塗るだけ塗ってテストしてみようか。」→「子供っぽくはなったんだけど，縦ロールというよりはドレッドヘアに見えなくもないなぁ…没にしようか？…でも見慣れてくるとこれはこれでありだなぁ…」その後2cの毛先を揃えたものを作成しようとトライアルしたものの，どうも上手くバランスが取れなくていくつか没（完成パーツ２個含む）にして，その間に2cのバランスを見慣れて来てしまったので採用としました。

2a，2bは明らかに同系統なので同じ番号のバリエーションで良いとして，2cは明らかにパターンが違うのでネーミングを「3」にしようかと思ったのですが，「まぁ同じ片側４本ロールだし」という事でバリエーション扱いとしました。

後から思えば明暗だけで処理すれば良い細部のディテール（髪の線）を描き込み過ぎたように思います。おかげでただでさえ手間のかかる作業が余計に手間がかかってしまって…


・縦ロールサイドアップa，b
使用フォルダ：hair_back，hair_back_accessory(aのみ)

縦ロール2cをツーサイドアップに束ねたイメージの髪型です。
おそらく頭の後ろはヘアピン挿しまくりではないかと。
そんなわけでリボンは必須だろうなとリボン装備としていますが，リボンなしバージョンもいつも通り（？）パーツ化しています。今回のリボンはものは試しにチェック柄としてみました。

縦ロール2cの輪郭線作成中に，上１本半分のロールを描いた辺りで，「この太さと位置なら，上にもう一巻き加えてツーサイドアップに出来そうだな」と思って後で弄るつもりで作業ファイルを別名保存しておいたものから作成してみました。

・三つ編みツーサイドアップ1a，1b，1c，1d，2a，2b（2a，2bはデフォルト改変）
使用フォルダ：hair_back，hair_back_accessory

ツーサイドアップのアップ部分を三つ編みにした髪型。
デフォルトパーツのセミロング2，ロング2が普通のツーサイドアップですね。

前々から作りたいな，とは思っていたのですが，三つ編みパターンの描き込み
を上手くやれる自信がないのと，面倒なのとでなかなか手を出せずにいたものです。

とりあえず1の方はデフォルトのツーサイドアップは意識しないで作成しました。で，デフォルトパーツを尊重してデフォルトのセミロング，ロングに三つ編み部分を重ねてセミロング2，ロング2の別バージョンみたいな扱いにしてみようかと思ったので，1a作成後にデフォルトのセミロングを重ねて作成したのが2aになります。

作成順は1a→2a→1b→1c→2b→1dです。
1a作成時に輪郭線の片側左右反転にて作成しているのですが，この時デフォルメ気味に広めに開いた三つ編みがなんかバランスが悪いような気がして向かって左側の三つ編みの角度を少し狭めた（1b）のですが，そうすると今度は右側も狭めた方がいいような気がして，それがいたちごっこになって両側共に狭めたものが1cです。で，これを決定稿にしようと思ったのですが…

2aはそのまんまでも良いような感じだったのですが，とりあえず決定稿にするつもりの1cに「まぁこの角度だとちょっと無理があるよな？」と確認作業でセミロングを重ねてみたわけですが，画面を拡大して作業をしていた関係で，貼り付ける位置が右方向へずれてしまったんですけど，この位置がかなり絶妙な位置で，1cの降ろした髪の上半分のラインとぴったり合ってしまったんですよね。しかも2aとずいぶん印象が変わっているという具合。
1cで角度を閉じた関係で前髪に半分隠れる右側の三つ編み上部がこぶに見えなくもないのが若干気になっていたので，「じゃあ（2bと）同じようにしてしまおう」と降ろした髪の部分を少し右側にボリュームアップしたものが1dになります。

そこでとりあえず全部テストして見較べてみたのですが，2aと2bはずいぶん印象が違うのでどちらも採用にするとして，角度の狭い1cと1dではなんとなくツーサイドアップらしくないような感じだけれども清楚な感じが出ているし，角度の開いた1aと1bはなんとなく元気っ娘っぽい感じでこれはこれで捨てがたいのでさぁどうしようか？となりまして…

ついでに様子見しながらあれこれ微調整で弄っていた関係でどれも捨てがたく，，結局見較べて悩んでいるのにも飽きたので，「もったいない」と全部投稿してしまう事にしました。1aと1b，1cと1dは製作者自身じゃないと「どこか違うの？」程度の差なんですけどねぇ…まぁ表情変化の表現にでも使ってやってください。

で，その過程で同じ三つ編みパターンの使い回しでロングも作るつもりでいたのですが，ちょっとした髪型違いでバランスが微妙になってくる経験をしてしまったので，「う〜ん，ロングにするのなら三つ編みも長くした方がいいな」と，今回はロングバージョンは未作成です。

そんなわけで，当初「どうせ使い回すのなら三つ編み部分をアクセサリーパーツにしてしまって汎用性を持たせるか？」という案もあったのですが，当然その案も引っ込めました。

なお，ファイル名に環境依存文字の半角カタカナを使用していますが，全角だと記号まで表示されないためなのでご容赦ください。

アクセサリーパーツ
・ヘッドドレスカチューシャ
使用フォルダ：accessory_front，accessory_back
「ロリータ風の縦ロールを作りたい」と思った所で考えたのがカチューシャ型か頭をすっぽり覆うタイプのヘッドドレス装備。でもひらひらはマウス入力では特にやりづらいんですよ。
そんなわけで後回しにしていたのですが，投稿時はせめてセットにしたい，と思っていたのでなんとか作成してみましたが，出来はいまいちなような。

頭をすっぽり覆うのはレイヤ分けの関係でかなり難易度が高いのでカチューシャタイプとしました。フリルのひらひらは何度もトライアルして思うように行かなかったので，布地を重ねるイメージのもので妥協しています。
当初汎用性を考えてもっと頭の上の方に配置していたのですが，かなりバランスが悪かったので今の位置に変更しています。おかげで合わない前髪パーツが結構あります。
色変化もさせるつもりだったのですが，同じくいまいち合わなかったので，急遽白黒に変更しています。白い部分を薄い色の変化としたので，黒い部分だけ変化させるようにすればあるいは良かったかも？

そんな試作レベルなので，おまけ扱いにでもしようかとも思ったのですが，流石に力尽きので，とりあえずのアップ。そのうち気が向いたらリトライするかもしれません。
