浴衣→着物変換パーツ　作成者：内野ねこ


◆はじめに

ダウンロードしていただき、誠にありがとうございます。
各パーツはキャラクターなんとか機の対応フォルダに入れてお使い下さい。


◆パーツについての注意

このパーツは以前配布した『浴衣セット』の「女性-浴衣」に対応したアクセサリーパーツです。
浴衣を着物（訪問着など）っぽくすることができます。
単体では使えませんのでご注意ください。


◆パーツ詳細

・アクセサリー
帯-太鼓(黄)1、(黄)2、(黒)1、(黒)2、(赤)1、(赤)2
帯-太鼓(瞳)1、(瞳)2、(髪)1、(髪)2、(服)1、(服)2
浴衣花柄3、3反転、4、4反転
浴衣-白半衿1、1反転、2、2反転


◆使用例
女性-浴衣(小)1+[浴衣-白半衿1+浴衣花柄3+帯-太鼓(黄)1]を使用しました。
頭部のパーツは別の場所で配布している「千歳リフ衣装パーツ」内の姫カット+三白眼です。


◆このセットの改変・改良について
改変・改良は基本的に自由にしてくださってかまいません。
改変・改良パーツの配布の際は、元パーツとして記述されていれば、連絡は必要ありません。


◆再配布について
以下の場合は再配布可とします。
・アップローダから流れ落ちてネット上から消えた場合
連絡は特に必要ありません。
