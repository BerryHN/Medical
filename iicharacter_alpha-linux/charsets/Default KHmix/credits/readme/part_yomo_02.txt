となっております。今回は前回に比べ少ないのでパーツについても少々。

・魔女帽子のベースはmieさん作の魔女帽子2です。
　改変に期待、と書かれていたので調子に乗ってしまいました(…)mieさんありがとうございます！
　そして、期待から外れた場合遠慮なく仰ってください…！可能な限り修正致します；

　あの帽子を拝見して、帽子の庇部分ではなく何か別のものでツインテールを隠せばどうだろう？
　と思い作ってみたのが魔女帽子4(でかリボンがついたやつ)です。
　2bは一応同梱しましたリボン無しです。
　mieさん作の帽子より画面左側の庇が若干長くなっております。
　＋アウトライン等をPhotoshopのシャープツールで際立たせたりしてみました。

　因みに、4でもヘアアクセの付いた髪型は隠せません。
　帽子被る時にあんなに大きなリボンで髪を纏めるのが悪い　という気持ちを込めました(ぇ)

　一応、ツインテール等のリボンの外し方ですが、リボンを外したい髪型を選択→コピーし、
　デスクトップ等に貼り付け、適当に名前を変更し(ex.ツインテール(長)リボン無 等)
　元のフォルダ(hair_back)に戻すだけです。
　yomoの説明じゃ分かんね　と思われましたら、BBSで質問すればきっと親切な誰かが答えてくださると思います^^

・ワンピースは、リクエスト版にて半袖のものを希望している方がいたので挑戦してみました。
　作成方法は一番上の通りです(…)ローブ＋体操着＋サマードレスで出来てます。

　体操服の袖口は日頃から何かに応用出来る　と思っていたので今回嬉々として貼り付けました。
　襟元がローブ引用なのは、フード付き風にしよう　と思ったためです。
　↑全てが余計だよ！！な感じの場合は…、えー、誰かが改変してくれるのを待ってくださいｗ(…)
