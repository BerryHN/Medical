振袖セット

・振袖とかんざしのセットです。

・振袖は柄物２種，格子柄，無地の２種類です。
　ver.2.xxのみの対応です。

・「柄物1」はどちらかと言うとテスト的なものです。
　試行錯誤で柄をなんとかしてみようとした結果，ランダムディザをかけてみましたが，影が目立たなくなるのが難点です。

・「無地」か「格子柄」で「body_front_color」を使用しないと，ピンク色無地の振袖になりますので，お好みでどうぞ。（添付のサンプル画像参照）
　柄物の「body_front_color」のパーツを見てもらえば分かると思いますが，柄の表現に「body_front_color」に消しゴムかけて「body_front」の下の色が透けるようにしてみたので，下地をピンクにしてみたんですよね。
　結局「柄物」の下地は色調に変化を持たせるために迷彩状態になっていますので，「柄物」を「body_front_color」を使用しないと，もの凄い絵になりますｗ
