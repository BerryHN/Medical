口紅パーツセット４３個です。表情にあわせてアクセサリーとして使うようになっています。

◆改変・参考元

◎参考パーツ
デフォルト及び、追加パーツ保管庫にある表情パーツ各種。
くちぶえ………雲之ツキ・パーツ内。（真夜様のブログにて配布）
「あ〜お」……重音テトパーツ内の口パク（キハ22様）
「あ〜お」N……南斗夏姫衣装パーツ内の口パク。（拙作）

申し訳ありませんが調べ切れませんでした。
パーツ製作の皆様、この場をお借りして御礼申し上げます。


◆このセットの改変・改良について

改変・改良は基本的に自由にしてくださってかまいません。
改変・改良パーツの配布の際は、元パーツとして記述されていれば、連絡は必要ありません。


◆再配布について

以下の場合は再配布可とします。
・UTAUシンガーキャラの素材としてまとめてUPする場合
・アップローダから流れ落ちてネット上から消えた場合
連絡は特に必要ありません。
