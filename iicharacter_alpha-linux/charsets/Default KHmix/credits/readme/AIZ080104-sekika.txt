石化パーツセット（ver2対応）
　リクエスト板のリクエストに，作成のアドバイスだけで済ませるつもりが，ずっぽりはまってしまって作成してしまったデフォルト改変パーツです。

・「石化頭」：「普通な顔1」ベースの改変の頭パーツです。
・「石化-1(肌)」「石化-2(肌)」：「素体1」と「素体2」ベースの改変で，アクセサリーパーツとして処理しました。
　頭と共に肌色の変更（褐色，日焼けに対応）が出来ます。

・「普通な目1（石化）」：「普通な目1」ベース改変の目パーツで，線画パーツなので瞳の色の変更には対応していません。
・「素2」：「素」ベースの改変（？）の表情パーツです。
　赤みが差した頬をとっぱらった状態です。「face_back」の方に頬があったので，単純に「face_front」だけにした状態なので，改変とも言えませんｗ

【使用上の注意】
　肌と目だけが石化しますので，当然髪と衣服は石化しません。色の違いは同系統の色で合わせてください。表情も赤みが差した頬やピンク色の口の中が見えないパーツを使用してください。
　質感の違いだけはいかんともしがたいので，必要と感じる方は自作をお願いします。

　石化パーツ作成の参考資料として，フリーのグラフィックツール，Pixiaを使用しての素体2の改変操作レポートをPDFで添付しておきました。

【作成に使用したグラフィックツール】
　pixia ver.4.2a
　マウス入力で作成
