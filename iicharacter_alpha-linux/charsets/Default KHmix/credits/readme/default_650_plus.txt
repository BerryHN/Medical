全身化素体派生集
デフォルト全身化セットを作る際にできた派生パーツのセットです。


ボトムス_サマードレス色付きはyomoさんのデフォルト色違いサマードレスと
セットで使える色変えパーツです。

ボトムス_ローブ黒はcutnateさんのローブ黒とあわせて使いたくて
作ってみた色違いパーツですが、ローブとしては短めになるので
スカート単体として使った方がいいかもしれません。

タイツとニーソはどの色にしようか迷いながら作っていたので
違う色のパーツがなんだかたくさんあります。ので全部入れました。
影を途中で変えたので縞用のパーツと合わない場合があります。
パーツ名に縞とついているものは、他の色のパーツに重ねて使う用です。

全身タイツはタイツがローライズ用でハイレグ等には対応してないので
長めで好きな長さに切って使えるタイツを作ろうとして、せっかくだから
全身用にした頭部パーツのないインナーです

なんとなく使った色メモ
(黒)　…デフォルト私服(シャツ)等で使われてる灰色っぽい黒
(黒濃)(服)　…デフォルトゴスロリ(ミニ)等で使われてる彩度低い濃いめの色
(濃)(服)　…デフォルトブレザー等で使われてる濃い目の色
(真黒)　…デフォルトブレザー等の濃い色をカラー選択で黒にした色
(白)(服)　…デフォルトサマードレス等で使われてる色変え用の白
(白)　…デフォルトセーラー服等で使われたり髪リボンにも使われてる白
(服)　…デフォルトエプロンドレス等で使われたり髪にも使われてる色
その他は過去に作ったもので使った色とか
